# Suppliers

Suppliers (vendors) represent the companies that supply your products. They are configured in the Merchello backoffice and own one or more [warehouses](./warehouses.md).

Suppliers primarily matter for multi-vendor stores. When vendor-based order grouping is enabled, orders are automatically split so each supplier receives only their portion of a multi-vendor order. Each group can have its own shipping options and be fulfilled independently.

For details on how vendor grouping works and how to enable it, see [Order Grouping](../shipping/order-grouping.md).
